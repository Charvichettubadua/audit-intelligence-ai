# 🛡️ Audit Intelligence AI: Real-Time Risk Analytics Dashboard

An enterprise-grade **AI-powered Audit Dashboard** designed to identify financial anomalies and generate forensic narratives using **Google Gemini 2.0/2.5 Flash**. This tool automates the detection of high-risk transactions and provides actionable insights for auditors.



## ✨ Key Features
- **Dynamic Risk Engine:** Automatically classifies transactions into Low, Medium, and High risk based on transaction amount and category.
- **AI Forensic Narratives:** Integration with **Google Gemini AI** to generate 3-point audit summaries for high-risk flags.
- **Interactive Visualizations:** Professional charts built with **Plotly** for risk distribution and exposure tracking.
- **Chrome-style UI:** Modern Glassmorphism dashboard interface built with **Streamlit**.

## 🛠️ Tech Stack
- **Frontend:** Streamlit (Python-based Web Framework)
- **AI Model:** Google Gemini 2.0 Flash (Generative AI)
- **Data Analysis:** Pandas, NumPy
- **Visuals:** Plotly Express
- **Environment:** Python Dotenv (for secure API handling)
